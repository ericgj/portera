= scheduler

== DESCRIPTION:

An application for determining the best time to conduct meetings.

Adapted from

* github.com/vertiginous/rmu-scheduler


== SYNOPSIS:

==== Basic usage

    schedule = Scheduler::Event.new do 
    
      week_of       Date.civil(2010,11,21)
      duration      90

    end
    
    schedule.participants << 
      Scheduler::Participant.new('Malcolm Reynolds').available do
        on(:monday,    :from => "15:00", :to => "18:00" )
        on(:wednesday, :from => "20:00", :to => "23:00" )
        on(:friday,    :from => "16:00", :to => "19:00" )
      end

    schedule.participants <<
      Scheduler::Participant.new('Hoban Washburne').available do
        on(:monday,    :from => "15:00", :to => "18:00" )
        on(:wednesday, :from => "20:00", :to => "23:00" )
        on(:thursday,  :from => "15:00", :to => "18:30" )
      end

    schedule.participants <<
      Scheduler::Participant.new('Jayne Cobb').available do
        weekdays(:from => "15:00", :to => "18:00" )
      end

    schedule.participants <<
      Scheduler::Participant.new('Zoe Washburne').available do
        on([:monday, :tuesday, :thursday], :from => "15:00", :to => "18:00" )
      end

    schedule.participants <<
      Scheduler::Participant.new('Inara Serra').available do
        on([:monday, :tuesday, :thursday])
      end
       
    puts schedule.best

==== Output

    Monday 15:00-18:00 +00:00
      Hoban Washburne
      Inara Serra
      Jayne Cobb
      Malcolm Reynolds
      Zoe Washburne
    Thursday 15:00-18:00 +00:00
      Hoban Washburne
      Inara Serra
      Jayne Cobb
      Zoe Washburne
    Tuesday 15:00-18:00 +00:00
      Inara Serra
      Jayne Cobb
      Zoe Washburne
